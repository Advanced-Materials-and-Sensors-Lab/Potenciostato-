# Open-Source Potentiostat for Environmental Analysis

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

An accessible, low-cost, and open-source potentiostat designed for educational and research applications. This project provides hardware schematics, firmware, and software to build and operate a functional potentiostat for electrochemical analysis.

## Overview

Commercial potentiostats are often prohibitively expensive for students and underfunded laboratories. This project aims to democratize access to electrochemical instrumentation. Currently, our team utilizes this device for environmental studies, specifically for the detection of heavy metals and microplastics.

**Project Status:** This project is currently in the prototyping phase. The electronic circuit is built on a breadboard, utilizing an Arduino Uno for data processing. We currently rely on LabVIEW for robust data acquisition and testing. Future updates will include a custom Printed Circuit Board (PCB) design and a dedicated Python-based graphical user interface (GUI) that meets international standards.

**Key Features:**
* Microcontroller: Arduino Uno.
* Low-Cost: Built with readily available electronic components.
* Software: Currently utilizing LabVIEW for data acquisition, transitioning to Python.
